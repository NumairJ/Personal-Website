Resume

Course Taken in TMU

Year One

CPS109 - Computer Science II

Introduction course that covers the fundamentals such as abstraction, modelling and algorithm design. Language used was Python.

CPS213 - Computer Organization I

Introduction to digital system designs that cover: Binary numbers, base conversions, signed numbers, Boolean algebra, logic gates, K-map method, combinational circuits, decoders/encoders, multiplexers, sequential circuits, flip-flops, state diagram, registers, counters and addressing techniques.

CPS209 - Computer Science II

A continuation of CPS109 with emphasis placed on code structure, algorithm development, and Object Oriented design principles. Language used was Jav.a

CPS310 - Computer Organization II

A continuation of CPS213 covering: Memory; CPU architecture and instruction set; the instruction processing sequence; generic assembler level programming; characteristics of major peripherals interfaces; RISC and CISC architectures compared; parallel processing. Lanuage used was Assembly.

Year Two

CPS305 - Data Structures

Introduction to commonly used data structures and algorithms. Data structures such as stacks, queues, lists, trees, and graphs. Algorithm topics such as searching, sorting, hashing, algorithm design, greedy approaches, dynamic programming, recursion and complexity analysis. Language used was Lisp.

CPS530 - Web Systems Development

Introduction to web services from its fundamentals and programming langauges for both server-side and client-side. Also covers responsive designs, search engine optimization and monetization strategies. Languages used for Front-end: HTML, CSS and Bootstrap, Javascript and jquery. Languages used for server-side: Perl, PHP, MySql, Classic ASP, Ruby and Python.

CPS393 - Introduction to UNIX, C and C++

Introduction to the Unix operating system and the C languages. UNIX topics include: I/O, redirection, processes, and shell scripts. C topics include pointers, structures, memory allocation, and paradigm differences. Not much C++ topics are covered. Languages used: Bash and C.
